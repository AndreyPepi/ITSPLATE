O objetivo do software é permitir a geração visual e dinâmica de placas de veículos para testes de leitura automática, em condições e adversas, visando validar sistemas de reconhecimento de caracteres.

- Principais Funcionalidades
Geração de placas aleatórias ou específicas
Formatos: Antigo (carro/moto) e Mercosul (carro/moto)
Renderização realista sobre imagens de placas de fundo
Imagens de fundo embutidas via Base64
Tamanho dinâmico da placa simulando aproximação/afastamento
Menu lateral animado com configurações rápidas
Cor de fundo personalizável
Cor de fundo variável (ciclando automaticamente)
Interface responsiva com design moderno (PySide6)
Compatível com compilação via PyInstaller para Windows e Linux
